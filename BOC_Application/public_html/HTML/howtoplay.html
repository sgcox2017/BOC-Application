How to Play

Summary

The purpose of this game is to give students an opportunity to study for the Board of Certification Exam with a fun and engaging game.

When pressing the "Play" button, the user will be given an opportunity to select a category or a random one. From there, the user will be prompted with a question to answer. They will have 20 seconds to answer the question. If answered correctly, the user will be awarded 10 points. Otherwise, they earn nothing. The user will then have an opportunity to select another category and question or return to the home screen.
